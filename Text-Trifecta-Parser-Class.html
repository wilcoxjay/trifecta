highlightToken :: TokenHighlight -> m a -> m a

highlightToken is called internally in the token parsers. It delimits ranges of the input recognized by certain parsers that are useful for syntax highlighting. An interested monad could choose to listen to these events and construct an interval tree for later pretty printing purposes.
